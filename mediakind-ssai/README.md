# THEOplayer React-Native MediaKind SSAI Connector

A MediaKind SSAI connector for `@theoplayer/react-native`.

## Installation

Using npm:
```sh
npm install @theoplayer/react-native-mediakind-ssai
```
[//]: # (npm install @theoplayer/react-native-mediakind-ssai)

Using yarn:
```sh
yarn add @theoplayer/react-native-mediakind-ssai
```
