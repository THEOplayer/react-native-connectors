# THEOplayer React-Native Clientside Ad Beaconing Connector

A Clientside Ad Beaconing connector for `@theoplayer/react-native`.

## Installation

Using npm:
```sh
npm install @theoplayer/react-native-clientside-ad-beaconing
```
[//]: # (npm install @theoplayer/react-native-clientside-ad-beaconing)

Using yarn:
```sh
yarn add @theoplayer/react-native-clientside-ad-beaconing
```
